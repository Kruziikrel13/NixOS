# Striking Distance
My primary workstation for gaming, university and work. Dual boots windows.

Uses btrfs for main drive.

### Specifications
- **Case:** Antec Flux Pro Case
- **Motherboard:** X870 Gaming X WIFI7
- **Keyboard:** Keychron Q6 ANSI Knob
- **Mouse:** Logitech G502 X

| Component | Specification |
|-----------|---------------|
| **RAM** | 32GB |
| **CPU** | AMD Ryzen 9800X3D @ 5.27 GHz |
| **GPU** | AMD Radeon RX 9070 XT |
| **Storage 01** (Main drive) | 500GB NVMe SSD |
| **Storage 02** | 2TB NVMe SSD |
| **Storage 03** | 2TB HDD |

### Notes
- Storage 02 has a games partition formatted using f2fs filesystem.
